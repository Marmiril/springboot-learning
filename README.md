# Spring Boot Learning

Proyecto de aprendizaje progresivo de Spring Boot usando Java, Maven y VSCode.

El objetivo de este repositorio es practicar los fundamentos de Spring Boot mediante ejercicios pequeños, separados y documentados. Cada ejercicio está organizado en su propio paquete para facilitar su lectura, mantenimiento y subida progresiva a GitHub.

## Tecnologías utilizadas

- Java 21
- Spring Boot
- Maven
- VSCode
- Spring Web

## Estructura del proyecto

```text
src/main/java/com/angel/springbootlearning/
├── SpringbootlearningApplication.java
└── exercises/
    ├── exercise01/
    │   └── Exercise01HelloController.java
    ├── exercise02/
    │   └── Exercise02PathVariableController.java
    └── exercise03/
        └── Exercise03RequestParamController.java


```




## Spring Boot Learning

Progressive Spring Boot learning project using Java, Maven and VSCode.

The goal of this repository is to practice Spring Boot fundamentals through small, separated and documented exercises. Each exercise is organized in its own package to make the code easier to read, maintain and upload progressively to GitHub.
